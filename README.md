## 📌 Overview

**NeuroLab Pro** is an interactive desktop application for understanding neural networks through **visualization and hands-on experimentation**. Perfect for students, educators, and AI enthusiasts.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🧠 **Perceptron Lab** | Single-layer neural network with AND/OR/NAND gates |
| 🔗 **MLP Lab** | Multi-layer perceptron with backpropagation |
| 💬 **Sentiment Analysis** | NLP text classification with confidence scoring |
| 🎨 **CNN Lab** | Convolutional neural networks with kernel visualization |
| 🔄 **RNN Lab** | Recurrent neural networks for sequence prediction |
| ⏳ **LSTM Lab** | Long Short-Term Memory with gate animations |
| 💾 **Hopfield Lab** | Associative memory for pattern storage/recovery |
| 📊 **Dashboard** | Real-time training analytics and progress tracking |

## 🎯 Key Capabilities

- ✅ **Interactive Training** - Adjust learning rate, epochs, datasets
- ✅ **Live Visualization** - Real-time accuracy/loss graphs
- ✅ **Neural Network Diagrams** - Animated architecture views
- ✅ **Educational Content** - Theory explanations + quizzes
- ✅ **Dark Theme UI** - Glassmorphism with neon effects
- ✅ **No GPU Required** - Runs on any computer
